## 插件
```
1. js语法降级支持
yarn add babel-polyfill -D  or npm i babel-polyfill -D
2.按需引入elementui
yarn add babel-plugin-component -D or npm i babel-plugin-component -D
yarn add element-ui or npm i element-ui
elementui循环项目中的svg，ant design不需要
3.清除console
yarn add babel-plugin-transform-remove-console -D or npm i babel-plugin-transform-remove-console -D
4.yarn add svg-sprite-loader -D  or npm i svg-sprite-loader -D
5.页面加载进度条
yarn add nprogress or npm i nprogress
5.axios请求
yarn add axios or npm i axios
```
